# Image Generation Guide for EcoPowerHub AI

This guide provides detailed specifications for generating all 57 required PNG assets.

## Visual Style Guidelines

### Color Palette
- **Primary Blue**: #0EA5E9 (sky-500) - Electric, modern energy
- **Primary Green**: #10B981 (emerald-500) - Sustainable, eco-friendly
- **Accent Teal**: #14B8A6 (teal-500) - Technology, innovation
- **Dark Background**: #0F172A (slate-900) - Professional depth
- **Light Accents**: #F0F9FF (sky-50) - Clean highlights

### Design Principles
1. **Futuristic Energy Network Theme**: Circuit-like patterns, glowing nodes, energy flow visualization
2. **Professional Dashboard Aesthetic**: Clean UI elements, data visualization, modern typography
3. **Transparency**: All logos and avatars must have alpha channel (no solid backgrounds)
4. **Consistency**: Maintain visual language across all assets in the same category

## File Requirements

### Technical Specifications
- **Format**: PNG with 8-bit color depth
- **Transparency**: Required for all logos and avatars
- **Compression**: Optimize to keep under 500 KB per file
- **Color Space**: sRGB
- **Resolution**: 72 DPI (web standard)

### Optimization Tools
- **TinyPNG**: https://tinypng.com/
- **ImageOptim**: https://imageoptim.com/
- **Squoosh**: https://squoosh.app/
